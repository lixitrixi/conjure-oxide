impl Expression

source

pub fn sub_expressions(&self) -> Option<Vec<&Expression>>

Returns a vector of references to the sub-expressions of the expression. If the expression is a primitive (variable, constant, etc.), returns None.

Note: If the expression is NOT MEANT TO have sub-expressions, this function will return None. Otherwise, it will return Some(Vec), where the Vec can be empty.

source

pub fn with_sub_expressions(&self, sub: Vec<&Expression>) -> Expression

Returns a clone of the same expression type with the given sub-expressions.
